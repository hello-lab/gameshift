# Glitch Wordle - Instruction Manual

## üéÆ Overview

**Glitch Wordle** is a challenging word-guessing game with progressive difficulty and glitch mechanics that make each word more unpredictable. Only team leaders can play and earn points for their personal accounts.

---

## üöÄ Getting Started

### Accessing the Game

1. **Log in** to your Gameshift account
2. **Navigate** to the **WORDLE** section from the main menu
3. If you're a **team leader**, you'll see the game board
4. If you're **not a leader**, you'll see: *"Leaders Only"* with the current leader's name

### Team Leadership

- Only **team leaders** can play Wordle
- Leaders earn points for their **personal user account** (not the team account)
- If you're not a leader, ask your team leader to play the game

---

## üéØ How to Play

### Basic Rules

1. **Guess the 5-letter word** in as few attempts as possible
2. **Enter words** using the on-screen keyboard or your physical keyboard
3. **Press ENTER** to submit your guess
4. After each guess, tiles show **color feedback**:
   - **Green** = Letter is correct and in the right position
   - **Yellow** = Letter is in the word but wrong position
   - **Gray** = Letter is not in the word

### Making a Guess

1. **Type** letters using the keyboard (on-screen or physical)
2. **Backspace (‚å´)** to delete the last letter
3. **Press ENTER** to submit when you have 5 letters
4. The game **auto-advances** to the next word after 1 second

### Game Flow

- You have **20 words total** to solve
- Each word has a **maximum number of guesses** (ranges from 8 down to 3)
- Words **increase in difficulty** as you progress
- After solving all 20 words, see your **final score and completion message**

---

## üíé The Glitch System

As you progress, glitches are introduced to make the game harder:

### Glitch Types

1. **COLOR SWAP** - A green or yellow tile randomly changes to the opposite color
2. **FAKE LETTER** - A gray letter is shown as yellow (it's not in the word)
3. **LETTER MUTATION** - A letter changes to a random different letter
4. **PHANTOM GREEN** - A gray tile is shown as green (it's not actually correct)
5. **REVERSE COLORS** - Green and yellow colors swap for that word
6. **INVISIBLE TILE** - A tile's feedback is hidden from you
7. **DELAYED FEEDBACK** - Feedback appears after a delay
8. **LENGTH SHIFT** - The word length is displayed incorrectly

### Progressive Difficulty

- **Words 1-3**: Minor glitches (0-2 glitches)
- **Words 4-6**: Medium glitches (2-3 glitches)
- **Words 7-9**: Chaos glitches (3-5 glitches)
- **Words 10**: Psychological glitches (5-6 glitches) - Most challenging!
- **Words 11-20**: Pattern repeats with increasing intensity

## üß† Strategy & Tips

### Tips for Success

1. **Start with common letters**: Q, U, E, A, R, S, T
2. **Look for patterns**: The glitch system hints at real feedback sometimes
3. **Don't trust colors fully**: Glitches can deceive you
4. **Use process of elimination**: Mark letters you've ruled out
5. **Try common word patterns**: -ING, -ER, -ED, TH-, ST-

### Dealing with Glitches

1. **PHANTOM GREEN**: If a green tile seems wrong, it might be a glitch
2. **FAKE LETTER**: A yellow letter might not be in the word
3. **COLOR SWAP**: Remember colors might be reversed
4. **DELAYED FEEDBACK**: Wait a moment for feedback to appear

### Planning Your Guesses

- **Guess 1**: Use a common word with 3 vowels (PLANE, STARE)
- **Guess 2**: Test different consonants based on feedback
- **Guess 3+**: Narrow down based on confirmed letters
- **Final guess**: Use remaining guesses strategically

## üÜò Troubleshooting

### Game Not Loading

- Check your internet connection
- Verify you're logged in as a **team leader**
- Try refreshing the page

### Keyboard Not Working

- Make sure the game window is **in focus**
- Try clicking on the game board first
- On mobile, scroll down to see the keyboard
- Try tapping keys instead of using physical keyboard

### Stuck on a Word

- You have limited guesses - if you run out, the game moves on
- You can't replay words you've already solved
- Come back with fresh eyes next time!

### Score Not Saving

- Check your internet connection
- The game requires an active session
- Refresh after playing to confirm scores were saved
